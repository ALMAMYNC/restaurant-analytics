# 🍝 Restaurant Analytics

## 🎯 Objectif du projet

Dans cette démonstration, nous jouerons le rôle d’un consultant en analyse de données pour un restaurant local. 🍝

Nous avons reçu des fichiers plats contenant des informations sur les plats du menu et les commandes des trois derniers mois, et notre mission est d’identifier des opportunités pour optimiser le fonctionnement du restaurant.

À l’aide de Python et Pandas, nous allons explorer les données et produire des visualisations permettant de mettre en évidence la saisonnalité des commandes, la performance des plats, et bien plus encore.

Au fil de l’analyse, nous suivrons les étapes d’une analyse exploratoire :
✔️ Lire et analyser les données provenant de fichiers plats
✔️ Joindre des données issues de plusieurs tables
✔️ Appliquer des méthodes de filtrage, de tri et d’agrégation
✔️ Visualiser les données à l’aide de graphiques et de diagrammes

---

## 🇬🇧 Overview

This project focuses on performing exploratory data analysis (EDA) on restaurant data using Python and Pandas.

We analyze menu items and customer orders to uncover insights such as:

* Order trends over time
* Best and worst performing dishes
* Sales patterns

---

## 🇫🇷 Description

Ce projet consiste à réaliser une analyse exploratoire des données (EDA) sur les commandes d’un restaurant à l’aide de Python et Pandas.

Nous analysons :

* Les tendances de commandes
* Les plats les plus et moins vendus
* Les performances globales du restaurant

---

## 📊 Datasets (Jeux de Données utilisées)

Les données sont sous forme de fichiers plats (CSV) contenant :

* Les informations sur les plats (menu)
* Les commandes des clients sur une période de 3 mois

---

## 🛠️ Technologies utilisées

* Python
* Pandas
* Matplotlib / Seaborn
* Jupyter Notebook

---

## 🔍 Étapes de l’analyse

1. Chargement des données
2. Exploration et compréhension des données
3. Nettoyage des données
4. Jointure des datasets
5. Analyse (groupby, aggregation, etc.)
6. Visualisation des résultats

---

## 📈 Exemples d’insights

* Les commandes augmentent pendant le week-end
* Certains plats génèrent la majorité des revenus
* Des plats sont peu performants et pourraient être retirés du menu
